## Content
-**block/**: this folder contains the depth-dose curve data for proton beams with different energies passing through a block made of 3D printing filament

-**rat_CT/**: this folder contains the _.dcm_ files which make up the CT scan of a 3D-printed rat skull

### Note:

For reasons concerning copyright and intellectual property, as well as privacy, the data has not been uploaded in its entirety
